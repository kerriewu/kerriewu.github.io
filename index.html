Remote-controlled robot

I designed and fabricated an RC robot that lifts two lanterns and hangs them about three feet above ground level. This robot won the Whitelaw Prize for Originality in Design and qualified for the final tournament with the top 32 scoring robots in a class of 150 students. This robot features an under-actuated three-stage linear lift which picks up two lanterns and also tilts them so that their holes align, allowing them to be placed on a single hook.

Smart Pill Bottle Construction Kit

I designed and wrote code for a “smart pill bottle” kit that modifies a standard pharmacy pill bottle as part of my work at Little Devices. The modified bottle tracks usage data whenever the bottle is opened and closed, and uses either an LED or pager motor as an alarm to remind the user to take a dose at specified times. This kit is intended for use by caregivers and patients who require medication, and for use in educational/outreach contexts.

Left Hand Free: A Glove-Powered Twitterbot

We made a prototype of a hand-controlled, wearable twitterbot that would tweet using text based on hand motions. Our prototype generated tweets by using sensors (light, tilt, and flex of the fingers) on the hand as a source of noise. The sensor readings are fed into a Markov chain text generator, based on a trigram model. This project was done in under 24 hours for HackMIT 2015. My team’s code can be seen on github (my username is kerriewu): link. A video demo can be found here: link.
